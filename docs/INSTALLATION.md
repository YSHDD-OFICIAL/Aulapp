# 📥 Guía de Instalación - CEA Biométrico

## Requisitos del Sistema

- Navegador moderno (Chrome 90+, Firefox 90+, Edge 90+, Safari 14+)
- Servidor web (Apache, Nginx, IIS) - opcional
- Node.js 18+ - para herramientas de desarrollo
- Conexión a internet (para CDN, opcional)

## Instalación Rápida

### Opción 1: Descarga directa

```bash
# Descargar el proyecto
wget https://github.com/tuusuario/cea-biometrico/archive/main.zip
unzip main.zip
cd cea-biometrico-main

# Iniciar servidor local
npx http-server public -p 3000
